## Tutorials Sub-Folder
===============================

This folder hosts the content for the tutorials that are published on the NEON website.

The tutorials are organized by programming language at this level. Most tutorials are written in R, followed by Python and Google Earth Engine (GEE). Additional resources are grouped under `Other`.

### High-Level Folder Diagram

```text
tutorials/                                         # Main folder for published NEON tutorials
|
|-- Other/                                         # Miscellaneous tutorials and setup resources
|   |-- Concept-intros/
|   |-- GEE/
|   |-- GEE_Python/
|   |-- HDF5View/
|   |-- Plasio/
|   |-- Software-Installation/
|   |-- Version-Control/
|-- Python/                                        # Python-language tutorials
|-- R/                                             # R-language tutorials
-- Tabbed/                                        # Bilingual tutorials with both R and Python tabs
```
