To complete this tutorial, you will need data available from the NEON 2017 Data Institute teaching data set available for download. Download NEON Teaching Data Subset: Data Institute 2017 Data Set The LiDAR and imagery data used to create this raster teaching data subset were collected over the National Ecological Observatory Networks field sites and processed at NEON headquarters. The entire dataset can be accessed by request from the NEON Airborne Data Request Page on the NEON website.
